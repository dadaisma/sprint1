Per executar l'exercici n1exercici3 des de la línia de comandos, seguiu aquests passos:


1. Obriu un terminal o una finestra d'ordres.
2. Navegueu fins al directori que conté el vostre codi font de Task5Maven\src\main\java\n1exercici3.
Utilitzeu la comanda 'cd' (change directory) per fer-ho.
3. Compileu el vostre programa Java amb la comanda 'javac', seguida del nom del fitxer font. Per exemple:

   javac Main.java

4. Després de compilar el fitxer, executeu-lo amb la comanda 'java',
 Per exemple:

   java Main.java

Amb aquests passos, l'exercici s'executarà des de la línia de comandos.
Assegureu-vos que esteu dins del directori correcte
que conté el fitxer 'n1exercici3.Main.java' abans d'executar les comandes.
Si tot és correcte, veureu la sortida de l'exercici al terminal.
